# SQL INSERT, UPDATE, DELETE and DROP TABLE Practice Task

This repository contains SQL practice tasks for database management using SQL Server.

The project includes two database systems:

* Company Management System
* Hospital Management System

## Topics Covered

* CREATE DATABASE
* CREATE TABLE
* INSERT INTO
* UPDATE
* DELETE
* DROP TABLE
* Primary Keys (PK)
* Foreign Keys (FK)
* Relationships between tables

## Tasks Included

### Company Management System

* Data Insertion
* Data Update
* Data Deletion
* Table Removal

### Hospital Management System

* Data Insertion
* Data Update
* Data Deletion
* Table Removal

## Files

* SQLQuery2_CompanyProject.sql
* SQLQuery_HospitalManagmentProject.sql
